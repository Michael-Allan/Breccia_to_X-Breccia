Breccia to X-Breccia Translator

This project implements a cursor-based pull translator of Breccia to X-Breccia. Taking parse states of Breccia and yielding those of X-Breccia, it makes Breccian text accessible to the full suite of JAXP, the Java API for XML Processing.
